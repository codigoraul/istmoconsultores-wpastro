# Istmo Consultores

Sitio web corporativo de Istmo Consultores - Consultoría estratégica en asuntos públicos, gobernanza y desarrollo de proyectos.

## Stack Tecnológico

- **Astro 4** - Framework web moderno
- **Tailwind CSS 3** - Estilos utility-first
- **TypeScript** - Tipado estático

## Estructura del Proyecto

```
├── public/
│   ├── images/          # Imágenes del slider
│   └── logo-ISTMO.svg   # Logo corporativo
├── src/
│   ├── components/      # Componentes reutilizables
│   ├── layouts/         # Layout base
│   ├── pages/           # Páginas del sitio
│   └── styles/          # Estilos globales
└── .windsurfrules       # Reglas de diseño
```

## Páginas

- `/` - Inicio
- `/nosotros` - Quiénes somos
- `/servicios` - Servicios
- `/equipo` - Equipo directivo
- `/sectores` - Sectores de intervención
- `/contacto` - Formulario de contacto

## Desarrollo

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build

# Vista previa de producción
npm run preview
```
